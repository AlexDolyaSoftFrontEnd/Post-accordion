# 📂 Posts-Accordion

Интерактивный React-компонент для структурированного отображения, редактирования и управления постами пользователей. 

---

## 🚀 Основные возможности

### 🔄 Загрузка данных

* Асинхронное получение постов из внешнего API (axios).
* Анимированный индикатор загрузки.
* Гибкая обработка ошибок (от отображения сообщения до повторной загрузки).

### 🧩 Группировка контента

* Автоматическая группировка постов по `userId`.
* Отображение первых 5 пользователей.
* Первый пост каждого пользователя разворачивается автоматически.
* Удобный аккордеон для просмотра всех постов выбранного пользователя.

### ⚙️ Интерактивные функции

* **Лайки** — возможность поставить/снять лайк.
* **Закладки** — сохранение постов в избранное.
* **Блокировка** — защита постов от редактирования.
* **Редактирование** — изменение заголовка и содержимого.
* **Всплывающие уведомления** — отображение результата действий (успех/ошибка) с fade-in и fade-out анимацией.

### 🛠 Работа с API (CRUD)

* Создание постов
* Обновление постов
* Удаление постов
* Перезапрос данных после операции
* Гибкая структура для подключения любого backend API

---

## 🖥 Пользовательский интерфейс

### 👤 Карточка пользователя

* Эмодзи-аватар, генерируемый по `userId`.
* Заголовок формата *User #{id}*.
* Бейдж с количеством постов.
* Анимация стрелки аккордеона.
* Стилизация на основе Bootstrap.

### 📝 Карточка поста

* Заголовок с эмодзи (например 📄).
* Основной текст поста.
* Бейджи активных действий: лайк, закладка, блокировка.
* Кнопки:

  * "Редактировать"
  * "Удалить"
  * "Лайк"
  * "Закладка"

### 🪟 Модальное окно редактирования

* Поля изменения заголовка и текста.
* Кнопки "Сохранить" и "Отмена".
* Поддержка disabled-состояний (например, при блокировке поста).
* Чистый Bootstrap-стиль + анимация появления.

## 🧑‍💻 Технические особенности

* **React Hooks:**
  Использование `useState`, `useEffect`, `useMemo`, `useCallback` для оптимизации.
* **TypeScript:**
  Чёткая типизация `Post`, `GroupedPosts` и пользовательских UI-состояний.
* **Axios:**
  Унифицированные запросы, возможность добавления interceptors.
* **Bootstrap 5:**
  Для сетки, модалок, кнопок и бейджей.
* **Собственные анимации:**
  Fade-out уведомлений, плавные переходы аккордеона.
* **Responsive Design:**
  Корректная адаптация интерфейса под мобильные устройства.
